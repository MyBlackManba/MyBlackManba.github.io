vps翻墙简单教程

让你快速看到外面的世界

最近vpn被封的厉害，很多以前快速实用的翻墙方法都不能使用了，连强大的goagent代理服务都没幸免。下面所要介绍的翻墙方法是基于vps的。

1.首先需要在购买一台国外的服务器。购买网址。大概2.5$每月。

第一个箭头可以查看用户名和密码，第二个可以远程连接国外服务器(这里也可以用PuTTY连接,文章最后会发所有工具的链接）

2.对服务器进行配置设置端口号，密码，加密方式等操作
登入putty登陆终端界面输入用户名以及密码开始配置
sudo vim /etc/shadowsocks/config.json（配置命令）
进入config.json后配置如下信息
{
“server”:”45.63.17.127”,
“server_port”:（自己定义建议8989以上）,
“local_port”:1080,
“password”:”自己的密码”,
“timeout”:300,
“method”:”aes-256-cfb”
}
netstat -pantu（用来查看服务器各端口进程）
ss-server -c /etc/shadowsocks/config.json（启动配置好的服务）
nohup ss-server -c /etc/shadowsocks/config.json（改为后台运行）
服务器配置完成

3.准备翻墙客户端————Shadowsocks.exe（文末链接有提供）
点击运行（简单吧！[手动滑稽]）

输完确定即可
