wifi干扰攻击

打打wifi也是挺好玩的

这个wifi干扰攻击是用airgedden搭建的基于软件的wifi干扰器。其原理是通过伪造所有链接在网关路由的主机mac地址
向路由发送请求断开的包。
攻击步骤（需要准备一个无线网卡）

1.下载airgeddon

命令行：git clone https://github.com/v1s1t0r1sh3r3/airgeddon.git
cd airgeddon
sudo bash airgeddon.sh

2.选择攻击网卡接口wlan0（填入前面的序号即可）

3.由于现在网卡处于managed模式，无法注入数据包需要将其改为监听模式monitor，输入前面的数字即可

4.扫面周边的接入点explore for target

5.选择好需要攻击的接入点后选择攻击方法mdk3 attack


6.回车即可
