php-数据库开发学习

学学后端！

phpstudy的一些功能简介
在其他选项菜单中选择网站根目录管理等等

一。站点域名管理：

1可以在mysql管理中启动。
2.为项目设置域名，可以通过域名来访问站点，域名和本地的网站目录进行绑定，从而可以创建多个目录。
3.vhost.conf可以看到相应的网站配置信息，虚拟主机配件
4.创建本地域名在host中ip加域名

二。mysql管理器进行数据库操作

1.mysqlfont
创建数据库，数据表，数据浏览来添加字段数据
2.phpmyadmin
同上

##　三。基于cms的网站后台管理搭建
1.先在本地搭建与域名绑定的网站文件根目录。
2.然后到www.phpcms.cn下载后台管理源码
3.将源码复制到网站根目录
4.进入配置的域名网址进行安装配置

四。基于thinkphp5的框架开发

1.先在本地搭建与域名绑定的网站文件根目录。
2.然后到thinkphp.com下载源码
3.网站根目录是要在public下

五。网站的根目录为APache配置文件中的documentroot项所指定的目录

1. mysql的配置。linux中配置文件通常为/etc/my.cnf,windows下的配置通常为mysql的安装路径下的my.ini文件
2.不使用GRANT而直接来修改授权表中的记录来改变权限，权限并不会马上生效，要使修改的权限马上生效可以在mysql提示符下执行
命令：flush privileges
3.mysql中数据备份的方法。
    方法一。命令：LOCK TABLES tablename  locktype(有read，write两种)(mysql界面下使用)
    方法二。命令：mysqldump -uroot -p123456 --opt --all-databases >direction dir(操作系统终端界面)
4.出错日志一般放在mysql的数据目录下面，使用编辑器打开查看即可。

六。一些数据库操作语句。

1.创建数据表语句
create table tablename(
列名 类型 一些规定语句(如auto_increment(自增))，
……
定义一下主键primary key
)
2.删除数据表 drop table tablename
3.模糊查询 select * from tablename where columnname like ‘%?%’ 来进行模糊查询
4.删除行 delete from tablename where culomn=?
5.创建用户并赋予其权力 GRANT privileges(增删改查) [columns] on item(具体表名) to username [identified by ‘password’]
6.插入语句 insert into tablename (col_name….) values (……)
7.更改数据表中的 update tablename set columnname=? [where=?]
8.更改数据表的结构 alter table tablename add columnname type(增加列)
alter table tablename drop columnname type(删除列)
9.修改列的属性 alter table users modify age char(10);
10.删除id自增长： alter table users change id id int;
11.创建用户 create user kobe identified by “root”;
12.查看用户权限：show grants for zz;
